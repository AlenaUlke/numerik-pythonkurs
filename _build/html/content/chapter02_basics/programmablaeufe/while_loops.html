How do we know how many times we need to roll the die? We dont. This is where a while loop comes in. A while loop repeatedly executes a chunk of code while a condition is true. If you know exactly how many time you need your code to be repeated, use a for loop. But if you want it repeated until some condition changes, use a while loop.
