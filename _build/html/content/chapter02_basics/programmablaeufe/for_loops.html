For-Schleifen#

Um die Idee hinter einer for-Schleife zu veranschaulichen, betrachten wir ein kleines Beispiel: das Kartenausteilen bei einem Spieleabend. Wenn Code eine feste Anzahl wiederholt werden soll, bzw. der selbe Code f√ºr verschiedene Werte, dann verwenden Sie eine for-Schleife.

Die Syntax#

Der Code im Schleifenrumpf einer for-Schleife wird so lange ausgef√ºhrt, bis alle Elemente des Iterationsbereichs - in diesem Fall \(\texttt{range(N)}\) vollst√§ndig durchlaufen worden sind.
