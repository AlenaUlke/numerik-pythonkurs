Funktionsaufrufe und -rückgaben#

Im Kapitel zur Arrayerstellung haben Sie bereits kurz die Funktionen shape, ndim und size kennengelernt, um Größe, Dimension und Anzahl der Elemente eines Arrays auszulesen.

  • np.shape(A): Größe des Arrays \(A\)

  • np.ndim(A): Dimension des Arrays \(A\)

  • np.size(A): Anzahl der gesamten Elemente des Arrays \(A\)

Anhand dieser Funktionen zeigen wir Ihnen wie Sie Funktionsrückgaben speichern und wie Sie mehrteilige Funktionsrückgaben in einzeln Varaiblen aufteilen können.

Show code cell content Hide code cell content 
import numpy as np

np.random.seed(42)
size = [np.random.randint(2, 10) for _ in range(3)]
A_mystery = np.random.randint(-20, 20, size=size)
B = np.random.randint(-20, 20, size=[3, 9])

Sie können die Funktion shape auf ein \(n\)-dimensionales Array anwenden, um eine einzige Ausgabevariable zu erhalten, die die Arraygröße in einem \(n\)-Tupel enthält.

Bei einem eindimensionalen Array entspricht der erste Eintrag der Anzahl der Elemente. Bei einem zweidimensionalem Array handelt es sich beim ersten Element um die Anzahl der Zeilen, während das zweite Element der Anzahl der Spalten entspricht.

Aufgabe 1.1

Erstellen Sie eine Variable mit dem Namen shape_B, die die Größe des Arrays B enthält.

print(B)
# Ihr Code

Hinweis

Verwenden Sie die Funktion shape mit der Variablen B.

Lösung

print(B)
shape_B = np.shape(B)

Aufgabe 1.2

Bestimmen Sie die Dimension, die Größe und die Anzahl der Elemente des Arrays A_mystery. Was fällt bezüglich der Beziehung zwischen der Größe und der Anzahl der Elemente des Arrays auf?

# Ihr Code

Lösung

print(np.shape(A_mystery))
print(np.size(A_mystery))
print(np.ndim(A_mystery))

Mehrteilige Funktionsrückgaben#

Sie können von der Funktion shape auch mehrere Ausgabevariablen anfordern. In diesem Fall enthält jede Variable die Größe einer Dimension des Eingabearrays.

xrow, xcol = np.shape(A)

Trennen Sie die einzelnen Variablen durch Kommata, um mehrere Ausgaben anzufordern.

Aufgabe 2.1

Erstellen Sie die Variablen num_rows und num_cols, die die Anzahl der Zeilen bzw. der Spalten der Variablen B enthalten.

# Ihr Code

Lösung

num_row, num_cols = np.shape(B)

Sie können die Eigenwerte und Eigenvektoren einer quadratischen Matrix \(A\) mithilfe der Funktion np.linalg.eig(A) bestimmen.

Die erste Ausgabe der Funktion liefert die Eigenwerte in Form eines Arrays und die zweite Ausgabe die zugehörigen Eigenvektoren in Form eines zweidimensionalen Arrays. Eine Spalte entspricht einem Eigenvektor.

Aufgabe 1.3

Erstellen Sie die Variablen ew_C und ev_C, die die Eignwerte und die entsprechenden Eigenvektoren von \(B\) enthalten.

C = np.array([[2, 0], [0, -4]])
# Ihr Code

Lösung

C = np.array([[2, 0], [0, -4]])
ew_C, ev_C = np.linalg.eig(C)

Wenn Sie nur eine Ausgabe einer Funktion nicht benötigen, können Sie diese Ausgabe ignorieren, indem Sie an ihrer Stelle einen Unterstrich (_) verwenden.

Beispielsweise können Sie in einer Matrix nur die Anzahl der Spalten bestimmen:

_, num_cols = np.shape(A)

Aufgabe 2.4

Erstellen Sie eine Variable mit dem Namen num_rows, die die Anzahl der Zeilen der Matrix A_mystery enthält.

Lösung

num_rows, _ = np.shape(A_mystery)
