Visualiserung#

Zur Visualisierung von Daten benutzt man in Python typischerweise das Modul pyplot des Pakets matplotlib. Daher importieren wir es zu Beginn dieses Kapitels:

import matplotlib.pyplot as plt 
import numpy as np

Vektoren in einem Diagramm darstellen#

Zwei Vektoren \(x\) und \(y\) derselben L√§nge k√∂nnen mittels der Funktion plt.plot(x, y) in einem Diagramm dargestellt werden. Der Vektor \(x\) entspricht dabei \(x\)- und der Vektor \(y\) den \(y\)-Koordinaten des Graphen.

x = np.array([1, 2, 3, 4, 5])
y = np.array([10, 11, 12, 15, 17])

plt.plot(x, y)
plt.show()

Der Befehl plt.show() wird in Python verwendet, um bei der Datenvisualisierung mit matplotlib das Diagramm anzuzeigen. plt.show() √∂ffnet ein Fenster (oder zeigt inline in Jupyter-Notebooks) das erstellte Diagramm an. Ohne diesen Befehl bleibt das Diagramm oft verborgen, besonders au√üerhalb von interaktiven Programmierumgebungen. Sowohl diese Website als auch Jupyther-Notebooks stellen eine interaktive Umgebnung dar. Hier k√∂nnen Sie den Befehl plt.show() also auch weglassen.

Beachten Sie, dass wenn Sie mehrere plot-Befehle direkt hintereinander ausf√ºhren ohne diese durch den Befehl plt.show() zu trennen, dann werden die Graphen entlang derselben Achsen geplottet in einen gemeinsamen Diagramm dargestellt.

Dieses Verhalten kann gew√ºnscht sein, muss es aber nicht. Um Python mitzuteilen, dass Sie beide Diagramme separat darstellen wollen, k√∂nnen Sie den bereits bekannten Befehl plt.show() verwenden oder den Befehl plt.figure(), welcher Python signalisiert, dass das Diagramm in einem neuen Plot mit eigenen Achsen dargestellt werden soll.

Im besten Fall verwenden Sie beide Befehle, denn plt.figure erstellt eine leere Leinwand (au√üerhalb von interaktiven Programmierumgebungen ein neues Fenster) f√ºr die kommenden plot-Befehle und der Befehl plt.show zeigt die Diagramme schlie√ülich an.

plt.plot(x, y)      # der Graph wir erstellt 
plt.show()          # der Graph wird angezeigt

plt.figure()
plt.plot(x2, y2)
plt.show()
