Exkurs: Transponieren eines Arrays#

Ihre innere Mathematikerin wird bei der Matrix-Vektor-Multiplikation vielleicht kurz stutzig geworden sein. Schlie√ülich muss \(x\) ein Spaltenvektor sein damit das Produkt \(A \cdot x\) wohldefiniert ist und ein Zeilenvektor, damit \(x \cdot A\) wohldefiniert ist. Das Ausf√ºhren von A @ x und x @ A funktioniert f√ºr eine quadratische Matrix \(A\) allerdings in beiden F√§llen problemlos. An dieser Stelle interpretiert numpy das Array x dynamisch entweder als Zeilen- oder Spaltenvektor.

F√ºr A @ x ist lediglich wichtig, dass die Spaltenanzahl von A gleich der L√§nge von x ist und f√ºr x @ A, dass die Zeilennanzahl von A gleich der L√§nge von x ist. Der Ergebnisvektor ist in beiden F√§llen wieder ein eindimensionales Array der entsprechenden L√§nge.
