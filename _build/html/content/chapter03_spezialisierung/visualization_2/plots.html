Arrayerstellung#

Ein numpy Array können Sie sich am einfachsten wie ein Vektor oder eine Matrix vorstellen. Es zeichnet sich durch drei Kernkenngrößen aus: die Dimension, die Größe sowie die Einträge des Arrays.

Bild

Die Dimension eines Arrays entspricht der Anzahl der Indizes, die benötigt werden, um ein Element zu adressieren. Beispielsweise kann ein eindimensionales Array als ein Vektor und ein zweidimensionales als Matrix interpretiert werden.

Die Größe eines Arrays (engl. shape) wird durch ein Tupel \((n_1, n_2, \dots, n_{\mathrm{dim}})\) charakterisiert, wobei \(n_i\) die Anzahl der Einträge der \(i\)-ten Dimension entspricht. Eindimensionale Arras haben die Größe oder Länge \(n\) und zweidimensionale die Größe \((m, n)\), was \(m\) Zeilen und \(n\) Spalten entspricht.

Manuelle Arrayerstellung#

Sie können ein eindimensionales numpy Array erstellen, in dem Sie dessen Einträge in einer Liste (gekennzeichent durch eckige Klammern []) zusammenfassen und der Funktion np.array übergeben:

# Einbinden des Pakets numpy unter dem Alias 'np'
import numpy as np 

x = np.array([0, 4]) # erstellt ein Array mit Einträgen 0 und 4
print(x)

Auch wenn ein eindimensionales numpy Array als Vektor interpretiert werden kann, ist es in Python weder ein Zeilen- noch ein Spaltenvektoren. In dem Sinne sind eindimensionale Arrays Objekte, welche Zahlen in einer festen Reihenfolge speichern. Im folgenden werden wir sie dennoch Vektoren nennen.

Sie können eckige Klammern verschachteln und durch Kommata trennen, um ein zweidimensionales numpy Array bzw. eine Matrix zu erstellen. Dabei geben Sie die Elemente Zeile für Zeile ein. Jede Zeile wird durch eckige Klammern abgegrenzt.

# erste Zeile von A: [1, 2, 3]
# zweite Zeile von A: [4, 5, 6]
A = np.array([[1, 2, 3], [4, 5, 6]])
print(A)

Vektoren mit Werten in gleichmäßigen Abständen#

In der Numerik und auch beim Programmieren (in Python) allgemein, benötigen Sie häufig Vektoren, dessen aufeinanderfolgenden Einträge immer die gleiche Differenz haben, wie zum Beispiel bei dem Vektor x:

x = np.array([3, 4, 5, 6, 7])
print(x)

# Ihr Code

Lösung

Bei langen Vektoren ist das manuelle Eingeben der einzelnen Werte äußerst unpraktisch. Eine schnellere Alternative zum Erstellen von Vektoren mit gleichmäßigem Abstand zwischen den Werten bietet die Funktion np.arange bei der lediglich der Start- und Endwert angegeben werden müssen:

np.arange(startwert, endwert)

Der Endwert ist allerdings nicht enthalten, das heißt np.arange erzeugt einen Wertebereich im Sinne des halboffenen Intervalls \([\texttt{startwert}, \texttt{endwert})\).

x = np.arange(3, 8)
print(x)

Aufgabe 2.2

Erstellen Sie mit dem Befehl np.arange einen Vektor mit den ganzzahligen Werten von \(10\) bis \(20\). Speichern Sie das Ergebnis in der Variable x_range.

# Ihr Code

Lösung

x_range = np.arange(10, 21)

Standardmäßig verwendet die Funktion np.arangeeinen Abstand von \(1\) zwischen zwei aufeinanderfolgenden Einträgen. Es ist aber auch möglich einen anderen Abstand festzulegen. Übergibt man der Funktion np.arange ein drittes Argument dann lassen sich auch Vektoren mit beliebigen Abständen erzeugen:

np.arange(startwert, endwert, abstand)

Zum Beispiel liefert der folgende Code ein Vektor mit Abstand \(3\):

x = np.arange(1, 13, 3)
print(x)

Anstatt den Abstand zwischen den einzelnen Elementen eines Vektors, ist es auch möglich die gewünschte Elementanzahl zwischen einem Start- und Endwert anzugeben. Dazu können Sie die Funktion np.linspace verwenden:

linspace(startwert, endwert, anzahl_elemente)

Im Gegensatz zu der Funktion np.arange ist der Endwert bei np.linspace im Wertebereich des Vektors enthalten. Die Funktion erzeugt also einen Wertebereich im Sinne des abgeschlossenen Intervalls \([\texttt{startwert}, \texttt{endwert}]\).

x = np.linspace(0, 1, 5)
print(x)

Arrayerstellung mittels Funktionen#

In Python gibt es neben Funktionen, die Vektoren mit aufeinander folgenden Werten oder mit Werten einer konstanten Differenz erzeugen, auch weitere Funktionen, mit denen Arrays beliebiger Dimension erstellt werden können. Zum Beispiel erstellt die Funktion np.random.rand(n) einen Vektor mit \(n\) zufälligen Einträgen.

x = np.random.rand(2) # erstellt ein Vektor der Länge 2
print(x)

Aufgabe 3.1

Erstellen Sie einen Vektor der Länge \(7\) mit Zufallseinträgen und speichern Sie den Vektor in der Variablen y_rand.

# Ihr Code

Hinweis

Verwenden Sie die Funktion np.random.rand und verwenden Sie \(7\) als Argument.

Lösung

y_rand = np.random.rand(7)

Die Funktion np.random.randn erlaubt auch mehrere Eingaben, um zum Beispiel Matrizen zu erzeugen. Der Code np.random.randn(n, m) erstellt eine \((n \times m)\) Matrix mit zufälligen Einträgen.

A = np.random.rand(3, 2) # erstellt eine 3 x 2 Matrix
print(A)

Aufgabe 3.2

Erstellen Sie eine \((4 \times 4)\) Matrix mit zufälligen Einträgen und speichern Sie die Matrix in der Variablen A_rand.

# Ihr Code

Lösung

Es gibt noch weitere Funktionen, um Arrays einer bestimmten Dimension und Größe zu erzeugen. Beispielsweise erzeugen die Funktionen np.zeros und np.ones Arrays, die ausschließlich aus Nullen bzw. Einsen bestehen. Die Erstellung von Vektoren funktioniert hierbei analog wie bei Funktion np.random.randn.

x = np.ones(7)
print(x)

Aufgabe 3.3

Erzeugen Sie mit Hilfe der Funktion np.zeros einen Vektor x_zero der Länge \(4\) und ausschließlich Null-Einträgen.

# Ihr Code

Lösung

x_zero = np.zeros(4)

Das Erzeugen von \((m \times n)\) Matrizen mittels np.ones oder np.zeros unterscheidet sich jedoch leicht zu der Funktion np.random.rand.

In diesem Fall muss man die Anzahl der Zeilen \(m\) und die Anzahl der Spalten \(n\) in einem Tupel (m, n) oder einer Liste [m, n] zusammenfassen und als Eingabe verwenden. Beides liefert jedoch dasselbe Ergebnis:

A_rund = np.ones((4, 2)) # runde Klammern um 4, 2
print("A_rund =", A_rund)

A_eckig = np.ones([4, 2]) # eckige Klammern um 4, 2
print("A_eckig =", A_eckig)

Aufgabe 3.4

# Ihr Code

Lösung

# alternativ
A_zero = np.zeros([4, 7])

Sie können die Größe, die Dimension und die Anzahl der Elemente eines bereits erstellten Array mit Hilfe der folgenden Funktionen auslesen:

  • shape: Größe des Arrays

  • ndim: Dimension des Arrays

  • size: Anzahl der gesamten Elemente des Arrays

A = np.random.rand(3, 5)

print(A.shape)
print(A.ndim)
print(A.size)

Mit nur einer Codezeile können Sie so ein Array erstellen, das genauso groß ist wie die bereits vorhandene Matrix.

B_same_shape_as_A = np.ones(A.shape)
