Plots mit Matrizen als Eingabe#

In diesem Kapitel lernen Sie, wie Sie Matrizen direkt als Eingabe f√ºr die Funktion plt.plot() verwenden k√∂nnen. Dadurch lassen sich mehrere Datenreihen gleichzeitig und besonders effizient in einem einzigen Plot darstellen.

Wir bleiben dabei bei dem bekannten Datensatz zum Anteil erneuerbarer Energien f√ºr die L√§nder Deutschland, USA, China, Norwegen und Brasilien, den Sie bereits im Kapitel Plots individuell gestalten kennengelernt haben.

import pandas as pd
import numpy as np

# Datenquelle
url = "https://raw.githubusercontent.com/owid/energy-data/master/owid-energy-data.csv"
df = pd.read_csv(url)

# Auswahlparameter
countries = ['Germany', 'United States', 'China', 'Norway', 'Brazil']
variable = 'renewables_share_energy'

# Daten filtern und auf relevante Spalten reduzieren
df_filtered = df[(df['country'].isin(countries)) & (df['year'] >= 1990)]
df_pivot = df_filtered.pivot(index='year', columns='country', values=variable)

# Nur vollst√§ndige Zeilen und relevante L√§nder
df_selected = df_pivot[countries].dropna()

# Jahr als eigene Spalte
df_selected_with_years = df_selected.copy()
df_selected_with_years.insert(0, 'year', df_selected.index)

# Umwandeln in NumPy-Array
data_array = df_selected_with_years.to_numpy()

Im Kapitel Plots individuell gestalten haben Sie gelernt, wie man einzelne Vektoren gemeinsam in einem Plot visualisiert. Nun gehen wir einen Schritt weiter: Statt jede Datenreihe einzeln zu √ºbergeben, k√∂nnen Sie ein 2D-numpy-Array direkt an plt.plot() √ºbergeben.

Wichtig: Die Dimensionen m√ºssen zueinander passen:

  • Der erste Parameter (die \(x\)-Achse) muss ein Vektor mit der gleichen L√§nge wie die Anzahl der Zeilen des 2D-Arrays sein.

  • Das 2D-Array enth√§lt dann pro Spalte eine eigene Datenreihe, die automatisch als separate Kurve geplottet wird.

In diesem Beispiel ist die Eingabe falsch: Der Vektor x hat die L√§nge 100, aber y besitzt nur 2 Zeilen und 100 Spalten.

y = np.vstack([np.sin(x), np.cos(x)])  # 100 Spalten, 2 Zeilen

plt.plot(x, y) 
plt.xlabel('x')
plt.ylabel('y')
plt.title('Sinus- und Cosinusfunktionen')
plt.show()

Nun korrigieren wir den Fehler: y hat 100 Zeilen und 2 Spalten ‚Äì passend zur L√§nge von x.

x = np.linspace(0, 2 * np.pi, 100)  # L√§nge 100
y = np.vstack([np.sin(x), np.cos(x)])  # 2 Spalten, 100 Zeilen

plt.plot(x, y.T) 
plt.xlabel('x')
plt.ylabel('y')
plt.title('Sinus- und Cosinusfunktionen')
plt.show()

Mit dieser Methode lassen sich also mehrere Kurven auf einmal zeichnen, indem Sie nur zwei Argumente √ºbergeben: den gemeinsamen \(x\)-Vektor und die Datenmatrix f√ºr die \(y\)-Werte.

Mit dem Befehl plt.axis() l√§sst sich die aktuelle Achsenskalierung abfragen oder gezielt setzen. Wird der Befehl ohne Argumente aufgerufen, gibt er die aktuell verwendeten Grenzen des zuletzt erzeugten Plots zur√ºck:

Dabei ist Folgendes zu beachten:
Damit plt.axis() die korrekten Achsengrenzen zur√ºckgibt, muss der Plot bereits vollst√§ndig gezeichnet worden sein. In vielen Umgebungen ‚Äì insbesondere in Jupyter Notebooks ‚Äì wird die Zeichnung jedoch erst durch den Befehl plt.show() ausgel√∂st. Nach diesem Aufruf ist der Zugriff auf die Achsengrenzen allerdings nicht mehr zuverl√§ssig m√∂glich. Verwenden Sie daher plt.draw(), um das Zeichnen vorab zu erzwingen. Erst danach gibt plt.axis() die tats√§chlich verwendeten Grenzen zur√ºck.Ein Aufruf von plt.axis() ohne Argumente gibt die aktuellen Achsengrenzen in folgender Reihenfolge zur√ºck: [x_min, x_max, y_min, y_max]. Das bedeutet: Die ersten beiden Werte stehen f√ºr die Grenzen der \(x\)-Achse, die letzten beiden f√ºr die \(y\)-Achse.

Wenn Sie die Achsengrenzen manuell festlegen m√∂chten, √ºbergeben Sie dem Befehl plt.axis() genau eine solche Liste: plt.axis([x_min, x_max, y_min, y_max]).
